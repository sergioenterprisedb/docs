## Uninstalling on Windows

1. From the Windows Control Panel, select **Uninstall a Program**.

1. Select **Migration Toolkit**. 

1. Select **Uninstall/Change**.

1. Select **Yes** to confirm the uninstall.

## Uninstalling on MacOS

Using the terminal, execute the following command. If necessary, provide a password when prompted. Superuser or administator privileges are required. 

```shell
sudo /Library/edb/mtk/uninstall-edb-migrationtoolkit.app/Contents/MacOS/installbuilder.sh
```
